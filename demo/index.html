🚀 Demos disponibles

Este proyecto demuestra la integración de AI Text Enhancer con varios editores WYSIWYG y Markdown populares:

  • Textarea Simple - Implementación básica con un textarea estándar
  • TinyMCE - Integración con TinyMCE incluyendo un botón en la barra de herramientas
  • CKEditor 5 - Integración con CKEditor con plugin personalizado
  • Quill - Integración con Quill editor
  • Froala - Integración con Froala WYSIWYG editor (requiere licencia)
  • StackEdit - Integración con StackEdit.js para edición avanzada de Markdown

Cada demo muestra cómo integrar el componente AI Text Enhancer tanto externo (fuera del editor) como interno (dentro de la barra de herramientas del editor). Para los editores Markdown, el contenido se mantiene en formato Markdown mientras se muestra como HTML en la vista previa.

Para comenzar:

  1. Introduce tu API key de OpenAI en el campo superior
  2. Haz clic en "Guardar API Key para todos los demos"
  3. Navega a cualquier demo usando los enlaces de arriba
